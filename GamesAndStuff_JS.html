Games And Stuff

Choose a game to launch.

Chess Game

Simple chess application for e-book readers, designed to make clumsy blunders to so anyone can beat it. If used on a Kindle Paperwhite load with ?kindle=1 at the end for an minor layout fix

Reversi Game

Classic Reversi/Othello with the same board-first layout and controls style as Chess (Depth, Clumsy, Preview and Undo), tuned for desktop, mobile, and e-reader style browsers.

Naughts & Crosses Game

Simple naughts-and-crosses game (five in a row). Also intended for e-book readers.

Mathbook Tool

Notebook-style JavaScript math tool with Run and Plot, X-range graphing, and helpers like hex/bin/dec/int plus sin/cos/tan/sqrt.

Calc Tool

Simple button-driven JavaScript calculator with expression entry, parentheses, 0x/0b input, A-F keys, and DEC/HEX/BIN result formatting.

Editor Tool

Just a simple editor in a webpage. To test different ways to access local files.

Rogue Game

A game like Rogue/Moria/Angband. Best played with keyboard. Find the ladder down to floor 10 and pick up the amulet. Move with arrow keys, cast spells 1-9. Or just click.

Thorium Gap Game

A dark vertical space shooter with a 3D craft, layered space effects, heavy combat, bosses, and mouse, touch, or button controls (under construction).

ShotEmUp Game

A whimsical shooter with a playful feel, built as an early prototype.

Soduko Joke

This is mostly a joke. Just leave this on in auto-mode over night and you've done your fair share of sodukos for this lifetime..

Tetris Game

Simple tetris game. Also intended for my e-book readers.

Mandelbrot Math

Very simple fractal program. Zoom in by dragging diagonally - from top left and down zooms in, other way to zoom out. Clicking somewhere also zooms in.

Ranger 2 Game

An attempt to use AI to generate interactive stories.
